# Immersive VR Learning – Interaction Scripts

This repository contains selected Unity C# scripts developed as part of my
Master’s thesis on immersive VR laboratories for autodidactic learning.

## Focus
- Task sequencing and validation
- Interaction constraints in VR
- Feedback mechanisms for procedural training

## Platform
- Unity
- XR Interaction Toolkit / OpenXR
- Standalone VR (Meta Quest)

Note: This is not a full Unity project.
Only core scripts relevant to interaction logic are included.
